# Noda 📚
**Hierarchical Knowledge Management & Smart Revision**

Noda is a professional-grade, distraction-free knowledge base and revision app designed for students and life-long learners. It combines the power of hierarchical folder-based organization with a clean, PC-style exploration experience and active recall features.

![Noda Logo](assets/Logo_Padded.png)

## ✨ Key Features

- **Hierarchical Explorer**: Organize your study materials in a professional folder-based system. Navigate through subjects, modules, and topics with ease.
- **Smart Reading Mode**: A premium, distraction-free reading experience with integrated **Text-to-Speech (TTS)** that automatically cleans Markdown and HTML for natural listening.
- **Active Recall & Spaced Repetition**: Integrated flashcards with a custom scoring system and real-time streak tracking to keep you motivated.
- **Universal Markdown Support**: Take notes using rich Markdown syntax, including highlights, task lists, and custom symbols.
- **PC-Style Navigation**: Dynamic breadcrumbs and parallel navigation between your Library and Knowledge Base.
- **Customizable Views**: Toggle between sleek List views and modern Grid-based "Knowledge Cubes."

## 🎨 Design Philosophy

Noda is built with a focus on **Visual Excellence** and **Premium UX**:
- **Modern Palette**: A carefully curated color system supporting both Vibrant Light and Deep Slate Dark modes.
- **Typography**: Optimized for long-form reading and quick scanning.
- **Micro-Animations**: Subtle transitions and hover effects that make the interface feel alive and responsive.
- **Adaptive Branding**: Custom-designed native splash screens and adaptive icons for a seamless boot-to-app experience.

## 🛠️ Technical Stack

- **Core**: Flutter (Dart)
- **State Management**: Riverpod (Reactive & Real-time)
- **Database**: Drift (SQLite) with reactive streams for instant UI updates.
- **Content**: Custom Markdown & Quill-based Rich Text.
- **TTS**: Flutter TTS with intelligent syntax stripping.
